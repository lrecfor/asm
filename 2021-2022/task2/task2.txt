1. Написать программу, считывающую строку (до нажатия клавиши Enter) в сегмент данных три раза подряд. Размер строки не больше 240 символов.

2. Написать программу, считывающую строку (до нажатия клавиши Enter) в сегмент данных. Размер строки не больше 240 символов. 
После вывести эту строку на экран. 
Использовать файловый сервис DOS (функции прерывания int 21h с номерами 0x3c-0x46).
Для считывания строки читать из стандартного ввода (дескриптор 0).
Для вывода строки писать в стандартный вывод (дескриптор 1).
